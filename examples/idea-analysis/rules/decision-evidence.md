# Decision evidence rules

- Every Fact must cite a sentence or constraint from the supplied brief.
- Distinguish facts from assumptions and state what would invalidate an
  assumption.
- A synthesis Intent must reference the pass Facts it combines.
- Reject recommendations that omit users, measurable outcome, feasibility,
  material risks or a validation experiment.
- EndFact must state go/pivot/no-go, the bounded pilot and the decisive metric.
